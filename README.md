## 🚀 Getting Started

1. **Create the files** in your project directory:
   - `index.html`
   - `styles.css`
   - `script.js`

2. **Open in browser**:
   ```bash

   open index.html
   ```

3. **Optional - Use a local server**:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

## 🎨 Customization Guide

### Change Brand Name
Replace "LUXE" in the navigation with your salon name:
```html
<span>YOUR_SALON_NAME</span>
```

### Update Contact Information
Edit the footer section:
```html
<p>📍 Your Address</p>
<p>📞 Your Phone</p>
<p>✉️ Your Email</p>
```

### Modify Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-gold: #D4AF37;
    --accent-purple: #8b5fbf;
    --accent-rose: #e8b4c8;
    /* Update as needed */
}
```

### Add Services
Add more service cards to the services grid in `index.html`:
```html
<div class="service-card">
    <div class="service-icon">
        <i class="fas fa-icon-name"></i>
    </div>
    <h3>Service Name</h3>
    <p>Service description</p>
    <a href="#" class="service-link">Learn More →</a>
</div>
```
